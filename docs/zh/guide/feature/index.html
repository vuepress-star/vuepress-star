vuepress-theme-star 引入了很多非常棒的功能。绝大部分功能都抽离成了单独组件，这意味着如果你喜欢它们，你可以在其他 VuePress 主题或默认主题中使用它们。

# 🧩 内建插件

本主题还包含了以下内建插件，如果有需要，你也可以单独进行使用或搭配其他主题。

  • @mr-hope/vuepress-plugin-comment: 评论与文章信息功能

  • @mr-hope/vuepress-plugin-component: 提供一些开箱即用的插件

  • @mr-hope/vuepress-plugin-copy-code: 提供一键复制代码块功能。

  • @mr-hope/vuepress-plugin-feed: Feed 支持

  • @mr-hope/vuepress-plugin-git: 基于 Git 的页面信息插件

  • @mr-hope/vuepress-plugin-pwa: PWA 支持

  • @mr-hope/vuepress-plugin-reading-time: 阅读时间与字数统计

  • @mr-hope/vuepress-plugin-seo: SEO 增强插件

  • @mr-hope/vuepress-plugin-sitemap: 为你的站点生成 sitemap

  • vuepress-plugin-add-this: 让网站支持社交分享与关注

  • vuepress-plugin-md-enhance: 提供更多 Markdown 语法

  • vuepress-plugin-photo-swipe: 使网站的图片支持缩放与滑动浏览
